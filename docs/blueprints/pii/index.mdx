# PII Blueprints

<Intro
  caption="Blueprints to help you meet Personal Data Protection regulations requirements"
  img={<Image />}
/>

Numerous Personal Data Protection Legislations (GDPR, CCPA, LGPD, PIPEDA, etc.) translate to effective regulative requirements for applications that collect, store and process sensitive data.

These Blueprints will provide you with guides to meet compliance requirements for specific use cases.
