### Create a Private Application

You will need a [Private Application](/docs/concepts/access-controls#basis-theory-account) to allow your backend to call Basis Theory APIs for Network Token operations. [Click here to create one](https://portal.basistheory.com/applications/create?permissions=network-token%3Acreate&permissions=network-token%3Acryptogram&permissions=network-token%3Asuspend&permissions=network-token%3Aresume&permissions=reactor%3Ainvoke&type=private) using the Basis Theory Customer Portal.

This will create an application with the following [Access Controls](/docs/concepts/access-controls):
- `network-token:create`
- `network-token:cryptogram`
- `network-token:suspend`
- `network-token:resume `
- `reactor:invoke`

<Alert type={Alerts.INFO}>
Network Token operations require Private Application permissions and are not available through Public Applications.
</Alert>
