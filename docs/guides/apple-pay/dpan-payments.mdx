<CreateApplePaySession />

Next, implement the `createApplePaySession` function.

```javascript
function createApplePaySession() {
  return new ApplePaySession(3, {
    "countryCode": "US",
    "currencyCode": "USD",
    "merchantCapabilities": [
      "supports3DS"
    ],
    "supportedNetworks": [
      "visa",
      "masterCard",
      "amex",
      "discover"
    ],
    "total": {
      "label": "Demo (Card is not charged)",
      "type": "final",
      "amount": "1.99"
    }
  });
}
```
