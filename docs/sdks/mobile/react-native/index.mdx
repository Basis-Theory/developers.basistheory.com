###

Basis Theory React Native is an [open source](https://github.com/Basis-Theory/basis-theory-react-native) [package](https://www.npmjs.com/package/@basis-theory/basis-theory-react-native) designed to allow you to quickly integrate Elements features in your Mobile application.

Utilizing these Elements is as simple as declaring them as a typical React Components and passing the required props.

```javascript showLineNumbers

  CardNumberElement,
```
