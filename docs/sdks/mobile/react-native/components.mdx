# Elements Components

## CardNumberElement

The `CardNumberElement` [component](https://reactjs.org/docs/components-and-props.html) features a card number input, by wrapping the [Card Number Element type](/docs/sdks/web/javascript/types#card-number-element) functionality and taking care of the React lifecycle.

<BasisTheoryComponentDisplay title="Live Card Number Element">
  <CardNumberElement id="types-card-number-element" />
</BasisTheoryComponentDisplay>

```javascript showLineNumbers
import {
  BasisTheoryProvider,
  CardNumberElement,
  useBasisTheory,
} from "@basis-theory/basis-theory-react";

const App = () => {
  // creates a new instance of BasisTheory class
  const { bt } = useBasisTheory("test_1234567890", { elements: true });

  return (
    <BasisTheoryProvider bt={bt}>
      <MyComponent />
    </BasisTheoryProvider>
  );
};

const MyComponent = () => {
  return <CardNumberElement id="cardNumber" />; // Element will grab instance from the Context
};
```
