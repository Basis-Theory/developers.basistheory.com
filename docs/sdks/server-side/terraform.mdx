### Installation

```hcl title="main.tf" showLineNumbers
terraform {
  required_providers {
    basistheory = {
      source  = "basis-theory/basistheory"
      version = ">= 0.7.0"
    }
  }
}

provider "basistheory" {
  api_key        = "<YOUR_API_KEY_HERE>"
}
```
