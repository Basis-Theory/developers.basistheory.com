## Installation

Add this dependency to your project's build file:

<Tabs className="bt-tabs" groupId="languages">
  <TabItem value="gradle" label="Gradle">
```groovy
repositories {
    mavenCentral()
    maven {
        url = uri("https://maven.pkg.github.com/Basis-Theory/java-sdk")
        credentials {
            username = 'GITHUB USERNAME'
            password = 'GITHUB PASSWORD or TOKEN'
        }
    }
}

dependencies {
    implementation 'com.basistheory:api-client:[version]'
}
```
  </TabItem>
  <TabItem value="maven" label="Maven">
```xml title="pom.xml"
<project>
  <repositories>
    <repository>
      <id>basistheory</id>
      <url>https://maven.pkg.github.com/Basis-Theory/java-sdk</url>
    </repository>
  </repositories>

  <dependencies>
      <dependency>
        <groupId>com.basistheory</groupId>
        <artifactId>api-client</artifactId>
        <version>[version]</version>
      </dependency>
  </dependencies>
</project>
```
```xml title="settings.xml"
<settings>
  <servers>
    <server>
      <id>basistheory</id>
      <username>[GITHUB USERNAME]</username>
      <password>[GITHUB PASSWORD or TOKEN]</password>
    </server>
  </servers>
</settings>
```
  </TabItem>
</Tabs>

The latest release version can be found in [GitHub](https://github.com/Basis-Theory/java-sdk/tags).

## Initialization

```java
import com.basis.theory.api.core.ClientOptions;
import com.basis.theory.api.resources.tenants.TenantsClient;

TenantsClient client = new TenantsClient(
                ClientOptions.builder()
                .addHeader("BT-API-KEY", "<API_KEY>")
                .build());
```
