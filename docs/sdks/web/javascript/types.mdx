# Elements Types

## Card Element

The `"card"` element type features a one-liner credit card collector containing the following inputs:

- *cardNumber*
- *expirationDate*
- *cvc*

<BasisTheoryComponentDisplay title="Live Card Element">
  <CardElement id="types-card-element" />
</BasisTheoryComponentDisplay>

```html showLineNumbers
<html>
  <body>
    <div id="card-element">

    <script defer>
      window.addEventListener('load', () => {
        const cardElement = BasisTheory.createElement('card');

        cardElement.mount('#card-element');
      });
    </script>
  </body>
</html>
```

## Text Element

The `"text"` element type is a secure replacement for the `<input>` tag and enables collecting user `string` data.

<BasisTheoryComponentDisplay title="Live Text Element">
  <TextElement id="types-text-element" placeholder="John Doe" />
</BasisTheoryComponentDisplay>

```html showLineNumbers
<html>
  <body>
    <div id="text-element">

    <script defer>
      window.addEventListener('load', () => {
        const textElement = BasisTheory.createElement('text', {

          placeholder: 'John Doe',
        });

    </script>
  </body>
</html>
```

## Card Number Element

The `"cardNumber"` element type renders a card number input and an optional card icon, featuring automatic brand detection, input validation
and masking.

<BasisTheoryComponentDisplay title="Live Card Number Element">
  <CardNumberElement id="types-card-number-element" />
</BasisTheoryComponentDisplay>

```html showLineNumbers
<html>
  <body>
    <div id="card-number-element">

    <script defer>
      window.addEventListener('load', () => {
        const cardNumberElement = BasisTheory.createElement('cardNumber', {
          targetId: 'elementTypesCardNumberElement',
        });

        cardNumberElement.mount('#card-number-element');

        cardNumberElement.on('change', ({ cardBrand }) => {
          window.cardVerificationCodeElement?.update({ cardBrand });
        });
      });
    </script>
  </body>
</html>
```
