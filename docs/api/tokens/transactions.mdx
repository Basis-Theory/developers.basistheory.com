# Transactions

## Create Transaction <Beta /> {#create-transaction}

Creates a new [transaction](/docs/concepts/transactions).

<HttpMethod
  method={HttpMethods.POST}
  endpoint="https://api.basistheory.com/transactions"
/>

#### Permissions

None

### Request

<Tabs className="bt-tabs" groupId="languages">
  <TabItem value="shell" label="cURL">

```shell showLineNumbers

  -H "BT-API-KEY: key_N88mVGsp3sCXkykyN2EFED" \
  -X "POST"
```

  </TabItem>
  <TabItem value="javascript" label="JavaScript">

```javascript showLineNumbers

const bt = await new BasisTheory().init("key_N88mVGsp3sCXkykyN2EFED");

```

  </TabItem>
  <TabItem value="csharp" label="C#">


## Transaction Object

| Attribute    | Type   | Description                                                                |
|--------------|--------|----------------------------------------------------------------------------|
| `id`         | _uuid_ | The transaction ID                                                         |
| `created_at` | _date_ | The date and time the transaction was created, in ISO 8601 format          |
| `created_by` | _uuid_ | The [Application](/docs/api/applications) ID which created the transaction |
| `expires_at` | _date_ | The date and time at which this transaction expires, in ISO 8601 format    |
