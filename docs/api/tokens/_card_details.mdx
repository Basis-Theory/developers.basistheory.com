### Card Details

| Attribute          | Type     | Description                                                                                                         |
|--------------------|----------|---------------------------------------------------------------------------------------------------------------------|
| `bin`              | _string_ | Six to eight digit BIN of the card                                                                                  |
| `last4`            | _string_ | Last four digits of the card                                                                                        |
| `expiration_month` | _number_ | The 2-digit expiration month of the card                                                                            |
| `expiration_year`  | _number_ | The 4-digit expiration year of the card                                                                             |
| `brand`            | _string_ | The primary [card brand](#card-brands)                                                                              |
| `funding`          | _string_ | The primary [funding type](#card-funding-types) of the card                                                         |
| `segment`          | _string_ | The segmentation of the card (eg., Consumer, Commercial)                                            |
| `issuer`           | object   | Describes the contry and issuing bank. See [Issuer](#issuer) for details                             |
| `issuer_country`   | object   |                                                                                                                     |
| _&nbsp;.alpha2_    | string   | Issuing country ISO3166 alpha country code                                                                          |
| _&nbsp;.numeric_   | string   | Issuing country ISO3166 numeric country code                                                                        |
| _&nbsp;.name_      | string   | Issuing country name                                                                                                |
| `authentication`   | _string_ | The [authentication type](#authentication-types) required for this card                                             |
| `additional`       | _array_  | Contains additional details associated to the same BIN number. See [Card Additional](#card-additional) for details. |

<Alert>`Card` properties shows the primary card details, while `Card.additional` provides additional card details found for the same BIN.</Alert>
<Alert>If a card number does not correspond to any known [BIN Details](/docs/api/tokens/token-enrichments#bin-details), then `brand`, `funding`, `issuer_country`, and `authentication` will default to `null` and the [default BIN](/docs/api/tokens/token-enrichments#default-bin) will be returned.</Alert>

### Card Additional
| Attribute        | Type     | Description                                                                       |
|------------------|----------|-----------------------------------------------------------------------------------|
| `brand`          | _string_ | An additional [card brand](#card-brands)                                          |
| `funding`        | _string_ | An additional [funding type](#card-funding-types) of the card                     |
| `authentication` | _string_ | An additional [authentication type](#authentication-types) required for this card |
| `issuer`         | object   | Describes the contry and issuing bank. See [Issuer](#issuer) for details                                                                                                                  |

### Issuer
| Attribute        | Type     | Description                                                                       |
|------------------|----------|-----------------------------------------------------------------------------------|
| `country`        | string   | Issuing country ISO3166 alpha country code                                                                          |
| `name`           | string   | Issuing bank name                        |
