Your Test Tenant can be configured to return `fake` or `real` data for the BIN Details enrichment. This change can be requested in the [Tenant's Quota Page](https://portal.basistheory.com/settings?tab=quotas).

When `fake` responses are enabled the following will occur:

1. If a card does not have a static value below, on every request a random set of BIN details following the `enhanced` data structure.
2. If a card does have a static value below, on every request the BIN details will be the same.

The following card numbers ranges have static BIN Details:

| Card Number Begin | Card Number End  | CVC          | Card Brand  | Test Card Example |
|-------------------|------------------|--------------|-------------|-------------------|
| 4242424242424242  | 4242424242424242 | Any 3 digits | VISA        | 4242424242424242  |
| 5555555555550000  | 5555555555560000 | Any 3 digits | MASTERCARD  | 5555555555554444  |
| 378282246310000   | 378282246310010  | Any 4 digits | AMEX        | 378282246310005   |
| 4000056655665550  | 4000056655665550 | Any 3 digits | VISA        | 4000056655665556  |
| 6200000000000000  | 6200000000000000 | Any 3 digits | UNIONPAY    | 6200000000000005  |
