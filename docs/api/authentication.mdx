# Authentication

Basis Theory uses API keys to allow access to the API.

Basis Theory requires the API key to be included in all API requests to the server in a header that looks like the following:

`BT-API-KEY: <API_KEY>`

<Alert>You must replace <code>&lt;API_KEY></code> with your API key supplied when you <a href="/docs/api/applications#create-application">created an Application</a>.</Alert>
